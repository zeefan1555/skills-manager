# RemNote 闪卡语法参考

本文件定义了 RemNote 能识别的所有卡片类型。卡片设计师读取本文件，
以确定为每张卡选择哪种分隔符。

---

## 卡片类型速览

| 类型 | 分隔符 | 最适用于 |
|---|---|---|
| 基础卡（正向） | `>>` | 简单问答、短定义 |
| 基础卡（反向） | `<<` | 反向回忆 |
| 基础卡（双向） | `<>` | 词汇对照 |
| 填空卡 | `{{...}}` | 在句子中回忆关键词 |
| 多行卡（正向） | `>>>` | 答案包含多个子要点 |
| 多行卡（反向） | `<<<` | — |
| 多行卡（双向） | `<><` | — |
| 列表答案卡 | `>>1.` | 有序序列 |
| 选择题卡 | `>>A)` | 考试备考 / 干扰选项 |

---

## 1. 基础闪卡

```
什么是有丝分裂？ >> 产生两个相同子细胞的细胞分裂方式
```

---

## 2. 填空卡

用 `{{...}}` 包裹需要隐藏的词。同一行有多个填空时，各自独立测试。

```
{{线粒体}}是细胞的能量工厂。
{{光合作用}}将{{阳光}}转化为化学能。
```

在 `}}` 后紧跟 `{({提示文本})}` 可添加提示：

```
{{柏林}}{({欧洲国家首都})}是德国的首都。
```

---

## 3. 多行卡  ← 本技能的主要类型

在问题行使用 `>>>`；所有缩进的子项目都会成为答案的一部分。

```
物质的三态是什么？ >>>
  - 固态
  - 液态
  - 气态
```

---

## 4. 列表答案卡

当顺序有意义时使用 `>>1.`。

```
按距太阳由近到远排列八大行星 >>1.
  - 水星
  - 金星
  - 地球
```

---

## 5. 选择题卡

使用 `>>A)`。**第一个子项目始终是正确答案**——RemNote 在复习时会自动打乱选项顺序。

```
金的化学符号是什么？ >>A)
  - Au
  - Ag
  - Fe
  - Cu
```

---

## 6. 额外卡片详情

在子项目行末尾追加 `#[[Extra Card Detail]]`——该内容会显示在卡片上，但不会被测试。

```
什么是 DNA？ >> 脱氧核糖核酸
  - 携带遗传指令的分子 #[[Extra Card Detail]]
```

---

## 缩进与格式规则

- 每行以 `- `（短横线 + 空格）开头。
- 每级缩进使用一致的空格数（2 或 4 个）。
- **卡片内部零空行。**
- **卡片之间恰好一个空行。**
- 分组标题与第一张卡之间不留空行。
